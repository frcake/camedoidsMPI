Χρήση προγράμματος

Compile with	:	make mpi
Run with	:	mpirun -np [number of processes] ./mpi_camedoids -f [dataset] -i data_centers -k [number of candidates] -c [number of clusters]


Για να λειτουργήσει η παραπάνω εντολή τα datasets & τα data_centers θα πρέπει να βρίσκονται στον ίδιο φάκελο με τον κώδικα.


Το πρόγραμα τρέχει σωστά με MPICH σε linux 

sudo apt-get install libcr-dev mpich mpich-doc
